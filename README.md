## Instalation Steps

1. Clone the repository
    ```
    git clone https://github.com/sbacanski0730/RIP-Rewak-and-PUM-Web.git
    ```
2. Go to project directory
    ```
    cd RIP-Rewak-and-PUM-Web
    ```
3. Install dependencies
    ```
    npm install
    ```
4. Run the app
    ```
    npm start
    ```
